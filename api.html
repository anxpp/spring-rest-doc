1、概要

本文为API说明文档。

API 分两部分，一部分为完全的 RESTFul 风格的 API ，另一部分为普通的 API

API的说明包括：请求方式、请求地址、请求参数说明和响应消息等。

2、通用说明

非 RESTFul API 通用响应消息说明

字段 类型 解释

state

int

1为成功，0为失败，其他为自定义状态

data

Object

请求的数据

msg

String

响应附带的消息

  "sex" : 1,
  "name" : "testAdd"
}'
